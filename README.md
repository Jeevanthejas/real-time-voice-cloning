# Real-Time Voice Cloning

This repository contains code and tools for real-time voice cloning.

Notes
- This project includes code that is covered by included `LICENSE` and related license files. Please read those license files for attribution and legal details.
- The repository was reinitialized locally; the git history was removed and replaced with a single initial commit.

Quick start
1. Install system tools and Python 3.8+:
   ```bash
   brew install ffmpeg
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. Run the demo:
   ```bash
   uv run --extra cpu demo_toolbox.py
   ```
